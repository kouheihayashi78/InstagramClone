# タイトル  
Instagram風アプリ

# 概要  
TECH I.S.の課題でTwitterCloneを作成した時、同じSNSであるInstagramを作成してみたいと考えていたので作成してみました。  
今まで作ったものよりも複雑なので、タスクや情報の関係性を洗い出し、テーブル設計を行いました。
